# AI GTM Discovery Agent

AI-powered GTM Discovery Agent that automates company research, stakeholder analysis, discovery question generation, executive summaries, and GTM briefing creation for sales and solutions teams.
## Live Demo

Website:
https://webpage-creation--archanakrishna4.replit.app

Loom Demo:
https://www.loom.com/share/19174ec0c8e245c792bb7ea6d23b8885

## Problem Statement

Sales Engineers and GTM teams spend significant time researching companies before discovery calls.

This AI agent reduces manual research by automatically generating:

- Company Overview
- Business Challenges
- Stakeholder Insights
- Demo Recommendations
- Discovery Questions
- Executive Summary
- Exportable Reports

## Features

- AI-powered company research
- Structured GTM briefing generation
- Stakeholder analysis
- Discovery question generation
- Demo preparation
- Report export
- Modern responsive UI
  

## Tech Stack

- React
- JavaScript
- Tailwind CSS
- AI APIs
- Emergent AI
- Replit

## Workflow

1. Enter Company Website or LinkedIn URL
2. AI researches the company
3. Business insights are generated
4. Discovery report is created
5. Export or download the report

## Business Value

This project helps GTM and Solutions teams:

- Reduce pre-call research time
- Standardize discovery preparation
- Generate stakeholder insights faster
- Improve meeting readiness
- Create reusable discovery reports
- Scale customer research workflows
